How To Use This Website

Use the drop down menu to select a cardiovascular drug class.

Each drug class has identified at least 1 cardiovascular drug that has been identified as teratogenic according to Briggs Drugs in Pregnancy and Lactation.

Once a class has been selected, the teratogenic drugs within that class are stated below the title

Only the drugs stated are what is being referred to as teratogenic. (Ex. Blood-thinners (Warfarin) only has teratogenic information relating to warfarin versus other blood thinners that have not been identified as teratogenic per Briggs)

The information stated in each drug class gives teratogenic information pertinent to pregnancy and breast-feeding.

The back arrow will take you back to the main menu to look up another class in the drop down menu.
